# Visual Style Guide

## Overall Mood

- 부드럽고 차분한 학생용 생산성 앱
- 기업형 대시보드보다 가볍고 친근한 분위기
- 여백이 넉넉하고 시선 흐름이 단순한 레이아웃

## Background

- 전체 화면에 파스텔 블러 배경 사용
- 좌측 상단은 연한 하늘색
- 우측 상단은 연한 핑크
- 하단은 라벤더와 샌드톤이 섞이는 느낌
- 배경은 강한 그라디언트보다 흐린 오로라 형태

## Surface

- 주요 카드와 패널은 반투명한 흰색
- `backdrop blur` 적용
- 큰 radius
- 그림자는 얇고 넓게 퍼지도록 설정

## Typography

- 본문은 깔끔한 산세리프
- 패널 타이틀은 세리프 계열로 포인트
- 제목 색은 채도 낮은 블루그레이

## Navigation

- 상단 내비게이션은 pill 형태
- 활성 탭만 진한 블루그레이 배경
- 비활성 탭은 흰색/반투명 배경 위 텍스트만 보이게 처리

## Home Page

- 저장공간 카드는 좌측에 작은 카드로 배치
- 원형 사용량 그래프는 굵고 단순한 링 형태
- 마감 임박과 즐겨찾기는 오른쪽 큰 카드에 리스트 형태

## Assignments Page

- 카드 안에 얇은 표 레이아웃
- 헤더 행은 작은 대문자 느낌
- 과목 라벨은 파스텔 pill 태그
- 제출 여부는 원형 토글 느낌으로 표현
- 즐겨찾기는 별 아이콘으로 표현

## Modal

- 모달 상단은 옅은 파스텔 그라디언트 헤더
- 폼 왼쪽, 업로드 영역 오른쪽의 2단 구성
- 입력창은 둥글고 밝은 배경
- 업로드 박스는 dashed border

## Admin Page

- 일반 사용자 공간과 같은 비주얼 톤 유지
- 다만 정보 밀도는 더 단순하게 구성
- 허용 사용자 관리가 목적이라 복잡한 관리 도구처럼 보이지 않게 유지
